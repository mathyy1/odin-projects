# landing-page

This is a simple landing page to train `CSS. It has a header, hero, info, blockquote, and footer sections.

It was a good practice with using flexbox, where I used it for pretty much everything. I also tried fixing up bad old habits of overusing relative units too much. 
I also learned about <blockquote> and <figures>, which was interesting. A lot of the time I had to ask myself, what should I use here instead of a <p>? Which led me to google it and learn stuff.
Overall, although I am more advanced than this, it was a good refresher and it helped me improve my fundamentals.
