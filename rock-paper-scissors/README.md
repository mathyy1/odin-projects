# Rock Paper Scissors Game

This is a simple Rock, Paper, Scissors Game. It is mainly created to practice DOM manipulation and logic.

While doing this project I learned a lot about how DOM works and how to get DOM elements. While the logic is quite simple,
reworking through a project was harder than I first thought, as I had to refactor quite a lot of the code so that it works
with the UI instead of the console. Creating the UI itself wasn't difficult, but understanding how to manipulate it through
JS showed quite difficult at the start. Overall, this was great practice and a learning experience for me. I learned about
DOM manipulation through querySelector, nested functions, and eventListeners.
